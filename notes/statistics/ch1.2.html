1.2. Probability Measures

A probability measure on \(\Omega\) is a function \(P\) from subsets of \(\Omega\) to the real numbers that satisÔ¨Åes the following axioms:

  • \(P(\Omega) = 1\)
    • since \(\Omega\) consists of all possible outcomes.

  • If \(A \sub \Omega\) , then \(P(A) \geq 0\)
    • meaning that a probability is nonnegative.

  • If \(A_1\) and \(A_2\) are disjoint, then \[P(A_1 \cup A_2) = P(A_1) + P(A_2)\] More generally, if \(A_1, A_2, ..., A_n, ...\) are mutually disjoint, then \[P\biggl(\cup_{i\ =\ 1}^{\infty} A_i \biggr) = \sum_{i\ =\ 1}^{\infty} P(A_i)\]

Properties of probability measures

1. \(\ \ P(A^c\ ) = 1 - P(A\ )\)
Proof: Since \(A\) and \(A^c\) are disjoint \[A \cup A^c = \Omega\] \[P(A \cup A^c\ ) = P(\Omega)\] \[P(A\ ) + P(A^c\ ) = 1\] \[P(A^c\ ) = 1 - P(A\ )\]

2. \(\ \ P(\empty) = P(\Omega^c\ )\)
Proof: \[\empty = \Omega^c\] \[P(\empty) = P(\Omega^c\ )\]

3. \(\ \ \) If \(A \sub B\ \), then \(P(A\ ) \leq P(B\ )\)
This property states that if \(B\) occurs whenever \(A\) occurs, then \(P(A\ ) \leq P(B\ )\)
Proof: \[B = A \cup (B \cap A^c)\] \[P(B\ ) = P(A \cup (B \cap A^c))\] \[P(B\ ) = P(A\ ) + P(B \cap A^c)\] \[P(A\ ) = P(B\ ) - P(B \cap A\ ) \le P(B\ )\]

4. \(\ \ P(A \cup B\ ) = P(A\ ) + P(B\ ) - P(A \cap B\ )\)
Proof: \[C = A \cap B^c\] \[D = A \cap B\] \[E = A^c \cap B\]

\[P(A \cup B\ ) = P(C\ ) + P(D\ ) + P(E\ )\] Also, \(A = C \cup D\) , and \(C\) and \(D\) are disjoint. So \[P(A\ ) = P(C\ ) + P(D\ )\] Similarly, \[P(B\ ) = P(D\ ) + P(E\ )\] Putting these results together, we see that \[P(A\ ) + P(B\ ) = P(C\ ) + P(E\ ) + 2P(D\ )\] \[P(A\ ) + P(B\ ) = [P(C\ ) + P(E\ ) + P(D\ )] + P(D\ )\] \[P(A\ ) + P(B\ ) = P(A \cup B\ ) + P(D\ )\] or \[P(A \cup B\ ) = P(A\ ) + P(B\ ) - P(D\ )\] \[P(A \cup B\ ) = P(A\ ) + P(B\ ) - P(A \cap B\ )\]
